Explicação de Atributos HTML para Formulários

  1. Atributo placeholder na tag <input type="text">

    O atributo placeholder exibe um texto temporário dentro do campo de entrada, orientando o usuário sobre o que deve ser inserido no campo.

  2. Atributo checked na tag <input type="radio"> ou <input type="checkbox">

    O atributo checked marca o botão de rádio ou caixa de seleção como selecionado inicialmente.

  3. Atributo value na tag <input>

    Define o valor inicial do campo, que será enviado quando o formulário for submetido, a menos que o usuário o altere.

  4. Agrupar botões de rádio

    Para agrupar botões de rádio e permitir que apenas um deles seja selecionado por vez, utilize o atributo name com o mesmo valor em cada <input type="radio">.

  5. Tag <label> e o atributo for

    A tag <label> associa um rótulo a um elemento de formulário. O atributo for deve corresponder ao valor do atributo id do elemento.

  6. Atributo selected na tag <option>

    Define uma opção como pré-selecionada em uma lista <select>, sendo a escolha inicial.

  7. Atributo required nas tags de formulário

    Indica que o campo é obrigatório e o formulário não poderá ser enviado se ele estiver vazio.

  8. Tag <textarea>

    Cria uma área de texto de várias linhas para inserção de textos longos, como comentários ou descrições.

  9. Atributo action na tag <form>

    Especifica a URL para a qual os dados do formulário serão enviados.

  10. Atributo method na tag <form>

    Define o método HTTP para envio dos dados do formulário. Os valores comuns são GET e POST.

  11. Método para proteger o valor da senha na barra de endereços

    Para garantir que uma senha não apareça na barra de endereços, o valor de method deve ser POST.

  12. Método para upload de arquivos

    Para fazer upload de arquivos, o method deve ser POST, e o enctype deve ser multipart/form-data.

  13. Tags <fieldset> e <legend>

    A tag <fieldset> cria um contorno ao redor de um grupo de elementos de formulário relacionados, enquanto <legend> adiciona um título ao grupo.
